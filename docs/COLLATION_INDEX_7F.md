### Ch.15 (51)(52) の系譜 (Boole Ch.XIX)

Boole Ch.XIX「統計的諸条件」の基本問題: n(xy) の上限 = min{n(x), n(y)}、下限 = n(x)+n(y)−n(1) または 0 — ケインズ (51) の「x/h+y/h−1<0 なら 0 で置換」注記まで class-count 版で対応。xyz 拡張 (minor = n(xy)+n(z)−n(1)) = (52) の階段。さらに**ブール自身の脚注が下限の初出を De Morgan に帰属** (coats/waistcoats 三段論法)。系譜: **De Morgan (三段論法) → Boole Ch.XIX (類の計数) → Keynes (51)(52) (確率関係形) → th_15_51/52 (カーネル認定)**。現代名 Boole–Fréchet 不等式 — Keynes 1921 は Fréchet 1935 に先行 (related work 節のネタ)。ケインズ訳注2の「Ch.XIX の解は不便で、後に Phil. Mag. 1854 で改良」という評も本文構成と整合。

**総括**: Ch.17 は「問題・特殊ケース・限界値・閉形式 = Boole 由来、一般解 ((57)(i)-(iii))・消去系 ((56.2)-(56.5), (57.1), (57.3))・仮定の精密化と方法批判 (fn.8/9) = Keynes 独自」という構造。Keynes は**ブールの全ての検算点 (根限界・特殊ケース・閉形式) を自前の計算体系で再現しつつ、ブールの方法 (論理方程式の展開法) を破棄**した — Phase 5/6a の形式化はこのケインズ側の再構築をカーネル認定したものであり、同時にブールの検算点 ((56.1)(57.2)(57.4)(57.5)(58.1)(58.3)) を区間・恒等式として回収している。Johnson 帰属 ((41)-(49)) と対をなす帰属地図として論文 §7 に反映する。7g 項目9への追記: (58.1) は Boole の "unless a=0" ガード付きで形式化 (0<a 前提)。
